## Instalación

```bash
python3 -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt
```

## Correr

### Generación de archivos

```bash
python main.py
```

Te genera los archivos:

- `campo_electrico.txt`.
- `campo_magnetico.txt`.

### Graficar

Usa el siguiente comando para observar las gráficas con matplot:

```bash
python graficar.py
```
